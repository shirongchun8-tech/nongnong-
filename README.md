# French Final Review

Static French study tool generated from `0610/Cours-RevisionFinale-Chinese-Review.md`.

## Use

```bash
npm run build
```

Open `index.html`, or deploy the repository root to GitHub Pages.

For GitHub Pages:

1. Push this repository to GitHub.
2. Open repository Settings.
3. Go to Pages.
4. Set Source to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

## Features

- Chapter-based French review with Chinese translations.
- Clickable French pronunciation using the browser's `speechSynthesis`.
- Slow playback for shadowing.
- Anki-like card review with local progress saved in the browser.
